# forum-app
Reddit-clone project
Tools used:
- TypeScript
- GraphQL
- URQL/Apollo
- Node.js
- PostgreSQL
- MikroORM/TypeORM
- Redis
- Next.js
- TypeGraphQL
- Chakra

# Cloning
You can clone the project and run the command<br>
```git update-index --assume-unchanged clone-server/src/mikro-orm.config.ts``` to keep your database credentials private
